# Lily the Foxgirl
Lily the Foxgirl is a Discord Bot who helps out around KyButler's server

## Current Features

- Auto-adds the 'Newbie!' role to any new user.
- !lily ping replies with 'Pong!'
- !lily avatar replies with the users avatar
